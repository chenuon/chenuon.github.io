    真假分辨

    为何如此一说，皆因目前市面上大多数软件都是收费的，我这里说的是电脑控制手机，很难找到免费的。
    我自己全部研究完毕，成功自建服务器后，发现确实不错，虽然有待改进，但现有功能够用了。

    支持哪些平台

    目前官网说的是全平台

    开始安装

    1、下载

    下载地址
    在这里选择最新版本的，我是windows下安装的，所以选择了windows平台的安装包，看不到的要吧那个列表下面那个show all xxxx点开才能看到windows的

    2、安装

    官网有详细安装教程，我是搬运工

    安装NodeJs

    请点击下载安装，可能会有点慢，如果卡顿太久，尝试关掉重新安装。 NodeJs是pm2的运行时环境，所以要先安装NodeJs。

    安装pm2

    在cmd.exe中分别输入下面三行，每一行都要按回车键，一行一行运行。 

    (这里我教大家一个快速简单的在当前目录运行cmd的方法，就是你进到某个目录，比如D:\123\,那么你在文件夹的地址栏直接点进去，吧地址栏内容删掉，直接输入cmd即可)

    npm install -g pm2
    npm install pm2-windows-startup -g
    pm2-startup install

    运行hbbr和hbbs

    这里用到上面**1、下载**的程序，解压到C以外的盘
    然后同样在这个文件目录内运行cmd
    分别运行下面四行命令
    pm2 start hbbs.exe
    pm2 start hbbr.exe
    pm2 save

    3、客户端配置

    上面运行hbbr & hbbs 程序之后，会生成一个.pub的文件，然后右键editplus打开，或者用什么其他软件都行，记事本也可以，我是习惯了。
    然后复制里面的内容添加到如下图的key里面，第一个和第二个输入你服务器的IP，本地电脑就127.0.0.1，局域网内其他电脑输入192.168.XXX.XXX

    确认就行，然后手机上面也下载APP，同样按照此方式配置，即可。
    手机里面把屏幕录制和输入控制开启，这样电脑链接的时候，手机确认接受一下请求即可。局域网还是比较快的。估计本机电脑更快，我是放局域网了。
