    云函数签到之天翼云盘(最新2023/3/8)

    技巧应用 签到 云函数 天翼云盘
    创建时间:2023-03-08 11:26
    字数:3.5k
    1. 工具
    2. 步骤
      1. 1，创建TX云函数
      2. 2，安装依赖包
      3. 3，添加配置文件
      4. 4，部署&测试

    近期折腾各大网盘，比如

    阿狸盘：2023/3/6开始已限速，准备放弃

    毒盘：虽说资源多，但不给米，你就别想用，故而一开始就没做主力

    移动：上传限速，下载我IDM可以跑10M，更大水管没试过

    联通：上传下载限速

    电信：上传下载不限，速度真的杠杠，空间小了点，所以最近折腾云函数签到拿空间，一天100M，一个月也有3个G。

    123：最近发现的这个，确实速度不错，但是有点担心走阿狸的后路。

    蓝凑：速度是不限，但是文件大小限制了，也要花银子。

    于是乎，针对此次天翼云盘的云函数签到，做了一番折腾

    工具

    工欲善其事必先利其器

    1，天翼云盘账号密码
2，TX云函数
3，push token         //消息推送，官网http://www.pushplus.plus 注册获取。

    步骤

    1，创建TX云函数

    选择从头开始

    index.py内容修改如下，

    # -*- coding: utf-8 -*-
from dailycheckin.main import checkin


def main_handler(event, context):
    checkin()

    然后其他按如下图操作即可，对了有个日志配置要启用，选默认即可，不然后面部署会报错。

    2，安装依赖包

    更新依赖与安装一致，输入如下两个命令即可：

    cd src
pip3 install dailycheckin --upgrade -t .
